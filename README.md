# 🌐 Auto Translator SAMP

![Status](https://img.shields.io/badge/Status-Active-brightgreen)
![Version](https://img.shields.io/badge/Version-1.0.0-blue)
![License](https://img.shields.io/badge/License-MIT-yellow)

A powerful and lightweight tool designed to bridge the language gap in **GTA San Andreas Multiplayer (SAMP)**. This application automatically translates in-game chat to help players communicate more effectively across different languages.

---

## 🚀 Features

* **Real-time Translation:** Seamlessly translates chat messages during gameplay.
* **Slang Support:** Optimized to handle common in-game slang and abbreviations.
* **Minimalist UI:** Designed to be non-intrusive and performance-friendly.
* **Easy Setup:** Compiled as a standalone `.exe` for quick deployment.

## 🛠️ Installation & Usage

1.  Download the latest release from the [Releases](#) section.
2.  Run the `Auto_Translator_SAMP.exe`.
4.  Launch your SAMP client and start communicating!

## 💻 Tech Stack

* **Language:** Python / C#
* **Tools:** Custom Translation Hooks, Automation Scripts.
